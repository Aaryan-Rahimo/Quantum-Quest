Displacement and Distance

Displacement is the change in position of an object, or how far an object is from its original position. It is a vector quantity, which means it has both magnitude and direction. Distance, on the other hand, is the total length an object has traveled from its original position, or the path it has covered. Unlike displacement, distance is a scalar quantity, meaning it only has magnitude and no direction.

Figure 1: Illustration of Distance and Displacement.

In Figure 1, the green dashed line represents the path taken by the object, which is its distance, while the straight black line represents the shortest path from point A to point B, which is its displacement. Even though the distance traveled is longer, the displacement only considers the straight line between the starting and ending points.

In this activity, we'll be using the distance sensor to explore the displacement of an object. The distance sensor will measure how far an object is from the sensor and output that distance in meters. This will allow us to observe the displacement of the object in real-time.
